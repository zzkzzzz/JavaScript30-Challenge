## NOTES
### CSS
    - `top`
      -   The top property affects the vertical position of a positioned element.
      -   `top: 100%;`  =>  `top: 0;`
        - set the top edge of element 100% down from the top edge of its nearest positioned ancestor
        - then set it to 0
        - which is animatable
